### <a name="getstarted"></a> Get started

Клонируем репозиторий
```console
git@github.com:SvyatFalse/avido-laravel.git
```

Билдим контейнеры
```console
make build
```
Запуск контейнеров
```console
make up
```
Остановка контейнеров
```console
make down
```
